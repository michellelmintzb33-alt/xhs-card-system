# XHS Card System — Design System

## 设计哲学

**保底好看** — 不是给用户更多自由度，而是收窄选择，确保下限。
产品替用户做设计决策，用户只负责提供内容。

---

## 双界面体系

| 界面 | 定位 | 用户 | 风格 |
|------|------|------|------|
| **maker.html** | 产品（封面制作器） | 内容创作者 | 浅色简洁 |
| **playground.html** | Studio（设计工作台） | 你自己 | 深色工具 |

## Maker UI 规格

### 布局

左右分屏：左 320px 控制面板 + 右自适应预览区。最小页面宽度 900px。

### 色彩

| 用途 | 色值 |
|------|------|
| 面板背景 | `#ffffff` |
| 页面底 | `#f8f8f8` |
| 标题文字 | `#111111` |
| 说明文字 | `#666666` |
| 占位符 | `#999999` |
| 主按钮背景 | `#111111` |
| 主按钮文字 | `#ffffff` |

基准单位 8px。常用间距：8 / 12 / 16 / 24 / 32 / 48。

### 字号

| 用途 | 大小 |
|------|------|
| 页面标题 | 20px / 600 |
| 区块标题 | 14px / 600 |

| 控件标签 | 13px / 400 |

### 圆角

- 按钮：8px

## 卡片模板设计系统

### 三色配色

| 变量 | 用途 | 用户可控？ |
|------|------|-----------|
| `--bg` | 卡片背景 | Maker: 是 / Studio: 是 |
| `--accent` | 主强调色 | Maker: 是 / Studio: 是 |
| `--sec` | 辅助色 | Maker: 是 / Studio: 是 |
| `--fg` | 正文颜色 | 自动推导 |
| `--muted` | 次要文字 | 自动推导 |
| `--border` | 边框/分隔 | 自动推导 |

### 推导规则

```
深色背景 (lum < 0.45):
  fg = #f0f0f0
  muted = mix(bg, #ffffff, 0.3)

浅色背景 (lum >= 0.45):
  fg = #111111
  muted = mix(bg, #000000, 0.38)
```

### 卡片尺寸

| 平台 | 尺寸 | 比例 |
|------|------|------|
| 小红书 | 1242 x 1660 | 3:4 |
| 抖音 | 1080 x 1920 | 9:16 |
| 微信 | 1080 x 1260 | 6:7 |

## 交互规范

### 导出按钮状态

```
默认:    [ 下载封面 PNG ]     #111 bg
加载中:  [ 导出中... ]        #111 bg, opacity 0.6, 禁用
成功:    [ ✓ 已保存 ]         #22c55e bg, 2秒后恢复
失败:    [ 导出失败，重试 ]    #ef4444 bg
```

### 空状态

- 首次访问：显示精选模板 + 示例内容的精美卡片
- 标题清空：显示灰色示例文字占位
- 图片未上传：显示上传区提示「点击上传背景图」

### 图片上传

- 最大 5MB
- 支持拖拽
- 超限提示：「图片太大，请压缩后重试（最大 5MB）」
