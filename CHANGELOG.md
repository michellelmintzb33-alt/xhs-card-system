# Changelog

All notable changes to the XHS Card System will be documented in this file.

## [0.1.0.0] - 2026-03-21

### Added
- 5 套视觉风格预设（韩系/黑金/潮流/柔和/瑞士）with color transforms and CSS overrides
- 5 个新模板：C29 极简图文风、C30 拼贴混乱风、C31 双色调摄影风、C32 手绘边框照片风、C33 自定义画布
- C33 自定义画布系统：图片上传、文字拖拽定位、滚轮缩放、blend mode 切换
- Gallery integration: 从 gallery.html 跳转自动加载模板和配色
- 系列保存/恢复支持 style 和 c33TextBlend 状态
- 导出系统支持 mix-blend-mode 属性和风格 class 传递
- DESIGN.md 设计系统文档（Maker UI 规格）
- TODOS.md 工程评审 TODO 清单
- CLAUDE.md 项目说明文档
